# Agildo Monitor Ultimate

Um monitor de sistema leve e moderno para Linux (focado em CachyOS/Arch), desenvolvido em Python e PyQt6.

## 🚀 Funcionalidades
- **CPU & GPU**: Monitoramento em tempo real (Suporte a AMD/NVIDIA/Intel).
- **Rede**: Velocidade de Upload e Download.
- **Processos**: Gerenciador de tarefas integrado.
- **Visual**: Interface moderna com tema escuro.

## 📦 Instalação
Clone o repositório ou baixe o executável na aba Releases.
